Walk-Throughs

These activities are not graded. Most tools introduced here are not mandatory. We appreciate, if you inform others about your preferred C++ coding practices.

  • Testing Machine on VirtualBox
    Install VirtualBox software. Download and run an Xubuntu instance (optionally - set it up on your own). Run a minimalistic C++ development environment. Check out, run and test some C++ projects from the command-line.
    Why do we care? Grading server uses "g++" and similar compilers in Linux; you can get advance knowledge, if something is not configured correctly. Automating your code to run on Xubuntu Linux can point to problems with your configuration; it will be also helpful to monitor and to compare running time of algorithms.
