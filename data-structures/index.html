The objectives of this course can be separated into a few big categories:

  1. Use C++ language features to implement basic algorithms. Use object-orientation to separate the public interface from the concrete implementation.
  2. Convert user stories into data manipulation calling ADTs. Use STL and similar predefined C++ data structure libraries.
  3. Discuss the implementation details for some popular data structures. Understand their complexity and tradeoffs in terms of time-space or performance-complexity.
  4. Introduce general paradigms of algorithmic thinking and revisit data structures along with algorithms to manipulate them as particular cases of these paradigms.
