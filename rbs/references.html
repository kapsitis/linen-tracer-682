(Hogbin Westby, 2015)
Hogbin Westby, E. (2015). Git for Teams. O'Reilly Media.
(Describes git and how to use it for teamwork. Custom git workflows and branching strategies. Code reviews, continuous integration, the roles of contributor, a reviewer, and so on.)
(Rubin, 2012)
Rubin, K. (2012). Essential Scrum: A Practical Guide to the Most Popular Agile Process. Addison-Wesley.
A thorough explanation of all Agile-related project practices and values. Scrum is one specific way to practice Agile methodology, which is most useful for technology related tasks that should quickly adopt to an environment with changing customer requirements and priorities.
